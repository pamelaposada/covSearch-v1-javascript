Welcome to CovSearch

Find Covid-19 Cases Around the World

Covsearch is a web application that allows you to search for Covid-19 cases around the world. By using Covsearch, you can get an updated Covid-19 overview for any country in the world, check their total cases, total deaths, vaccination status, last 30 days cases, etc. You only need to start a new search and type the country name that you want to search.
